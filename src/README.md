## 客户端安装

### 安装环境

```
# 安装 vue-cli 和 脚手架样板代码
npm install -g vue-cli
vue init simulatedgreg/electron-vue my-project

# 安装依赖并运行你的程序
cd my-project
yarn # 或者 npm install
yarn run dev # 或者 npm run dev

#安装其它依赖
yarn add iview
yarn add axios
yarn add vue-axios
yarn add vue-socket.io
yarn add buffer
yarn add js-base64
yarn add js-sha256
yarn add echarts
yarn add vue-echarts-v3
yarn add vue-i18n
yarn add fb
```

### 打包

```
yarn run dev  //debug
yarn run build //打包
```
